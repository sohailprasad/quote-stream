Quote Stream

Push real-time stock quotes in JSON over web-sockets with node.js.

Download this project as a .zip file Download this project as a tar.gz file

Live Demo

Tweet

Apple Inc. (AAPL)

http://xhv51cz0.nodesocket.com/quote-stream/AAPL/

Simply replace AAPL in the above URL with a valid ticker symbol. Currently, Quote Stream only supports passing a single ticker symbol. Stock quotes are pulled from Google Finance and are real-time.

Additional Details

By default stock quotes are fetched every 5 seconds, and pushed to the client. The fetch interval can be adjusted, and if you want the JSON response pretty-printed or not.
